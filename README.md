# Etch-a-Sketch (Browser Version)

An interactive Etch-a-Sketch web app featuring dynamic grid resizing and hover-based drawing, built with vanilla JavaScript to practice DOM manipulation and UI logic.

Built as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.  
See: [Etch-a-Sketch project](https://www.theodinproject.com/lessons/foundations-etch-a-sketch)

## Overview

This project focuses on building an interactive grid-based drawing tool in order to strengthen understanding of JavaScript-driven UI updates and dynamic DOM manipulation.

Users can interact with a grid to create simple drawings directly in the browser.
